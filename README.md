# Data-Science-Internship
Machine learning projects completed during a Data Science Internship, covering classification and prediction tasks using Scikit-learn, Pandas, and Matplotlib.

📊 Data Science Internship:
This repository contains machine learning projects completed as part of my Data Science Internship. Each project covers a different core ML concept, from classification to regression, with full data analysis, feature engineering, model training, and visualization.

📂 Projects
1. 🚢 Titanic Survival Prediction
Predicts whether a passenger survived the Titanic disaster using classification models (Logistic Regression, Random Forest, Gradient Boosting, SVM). Features include passenger class, sex, age, fare, family size, and extracted title. Achieved ~100% CV accuracy on the labeled test set.
2. 🎬 Movie Rating Prediction
Predicts IMDb ratings for Indian movies using regression techniques. Engineered features from director, lead actors, genre, votes, and duration. Gradient Boosting achieved the best performance with CV RMSE of 0.6788 and R² of 0.7584 on 7,900+ movies.
3. 🌸 Iris Flower Classification
Classifies Iris flowers into three species (Setosa, Versicolor, Virginica) based on sepal and petal measurements. Compared 6 classifiers with 10-fold cross-validation. SVM with RBF kernel achieved the best test accuracy of 96.67%.
